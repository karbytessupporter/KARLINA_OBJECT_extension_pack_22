This C++ program demonstrates RSA (Rivest–Shamir–Adleman) encryption and decryption.

--------------------------------

STEP_0: Key Generation

--------------------------------

q = 7393 // second prime number

n = p * q = 39249437 // the modulus in both the encryption and the decryption processes

phi = (p - 1) * (q - 1) = 39236736 // Euler's Totient function: ϕ(n) = (p − 1) * (q − 1)

--------------------------------

STEP_1: Choose e such that 1 < e < phi and gcd(e, phi) = 1

--------------------------------

e = 1566559 // public exponent

--------------------------------

--------------------------------
